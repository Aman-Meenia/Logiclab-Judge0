# Longest Palindrome Substring

&nbsp;

#### Given a string s, find the longest palindromic substring in s.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
s = "abccada"
```

&nbsp;

**Output**

```
ada
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
s = "madam"
```

&nbsp;

**Output**

```
madam
```

&nbsp;

**Example 3:**

&nbsp;

**Input**

```
s = "nnaannadanna"
```

&nbsp;

**Output**

```
annadanna
```

&nbsp;

### Constraints

&nbsp;

- 1 <= s.length <= 10^5

  &nbsp;

- s consists of only lowercase English characters.

  &nbsp;
