# Count Number Of Subarray With Sum Less than Or Equal To K

&nbsp;

#### Given an array arr[] of size n. The problem is to count the subarrays with sum less than or equal to k.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
arr = [-1, -2, 1, 2 ,-3]
```

&nbsp;

```
k = -2
```

&nbsp;

**Output**

```
6
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
arr = [3,-1,3,9,-8,1,-12,1,-3,4,-1]
```

&nbsp;

```
k = 3
```

&nbsp;

**Output**

```
55
```

&nbsp;

**Example 3:**

&nbsp;

**Input**

```
arr = [1,1,1,-1,1,1]
```

&nbsp;

```
k = 1
```

&nbsp;

**Output**

```
11
```

&nbsp;

### Constraints

&nbsp;

- 1 <= arr.length <= 10^5

&nbsp;

- -10^9 <= arr[i] <= 10^9
